Interactive calculator — target Hz → TICK_MAX

System clock is fixed at 100 MHz. Type the tick rate you want; it updates live and gives the TICK_MAX parameter, the counter width, and the real rate after integer rounding.

Formula: tick pulses one clock every TICK_MAX cycles, so rate = 100 MHz / TICK_MAX → TICK_MAX = 100,000,000 / target Hz. The counter holds up to TICK_MAX-1, so it needs ceil(log2(TICK_MAX)) bits — widen cnt past [26:0] below ~0.75 Hz.

\subsection*{Simulation --- VGA\_Interface}

The design targets the \textbf{800$\times$600 @ \SI{72}{\hertz}} VGA mode driven by a
\SI{50}{\mega\hertz} pixel clock, generated by dividing the board's
\SI{100}{\mega\hertz} system clock by two. Two counters track the beam position:
\texttt{h\_count} runs \numrange{0}{1039} (\num{1040} pixel-clocks per line) and
\texttt{v\_count} runs \numrange{0}{665} (\num{666} lines per frame). Only the
\num{800}$\times$\num{600} window is visible; outside it the outputs are blanked to
black. % TODO: expand with your own wording

\textbf{Signals} (top to bottom in the waveform):
\begin{itemize}[noitemsep,topsep=2pt,leftmargin=1.4em]
  \item \texttt{clk} - \SI{100}{\mega\hertz} system clock (test-bench driven).
  \item \texttt{rstn} - active-low reset.
  \item \texttt{pclk} - internal \SI{50}{\mega\hertz} pixel clock (clk divided by two).
  \item \texttt{pixel\_color[11:0]} - input colour word \{R,G,B\} from the Drawer.
  \item \texttt{XCoord[10:0]} / \texttt{YCoord[10:0]} - current pixel position.
  \item \texttt{Hsync} / \texttt{Vsync} - sync outputs (positive polarity).
  \item \texttt{vgaRed/Green/Blue[3:0]} - colour outputs (forced to 0 in blanking).
\end{itemize}
